1. 项目摘要

随着互联网信息的爆炸式增长，用户面临着信息过载的问题。推荐系统作为解决这一问题的有效手段，已经广泛应用于电子商务、社交网络等领域。本项目旨在构建一个基于协同过滤与深度神经网络相结合的混合推荐模型...

在本研究中，我们首先对MovieLens数据集进行了清洗和预处理，然后分别实现了UserCF和ItemCF算法作为基准。随后，引入了DeepFM模型来处理高维稀疏特征...
