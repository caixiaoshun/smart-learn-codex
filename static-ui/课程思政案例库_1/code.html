description 案例背景与描述

中医药是中华民族的瑰宝，其中蕴含着丰富的经验智慧。然而，传统的传承方式往往依赖于口传心授，缺乏系统性的定量分析。随着大数据技术的发展，我们可以利用现代数据挖掘技术对海量的中医医案、处方进行分析，揭示药物之间的配伍规律，为现代新药研发提供启发。

本案例选取了著名的《伤寒杂病论》中的部分经典方剂作为数据集。学生将扮演数据分析师的角色，任务是利用关联规则挖掘算法（如 Apriori 或 FP-Growth），找出高频出现的药物组合（项集）以及强关联规则。

核心问题：

  • 哪些中药材经常一起出现？（频繁项集挖掘）
  • 当出现“桂枝”时，有多大概率会出现“甘草”？（关联规则置信度分析）
  • 如何解释这些规则背后的药理学意义？（领域知识结合）

通过本案例，学生不仅能掌握关联规则算法的原理与实现，还能深刻体会到“古为今用”的创新思维，增强对民族文化的认同感和自豪感。

import pandas as pd
from mlxtend.preprocessing import TransactionEncoder
from mlxtend.frequent_patterns import apriori, association_rules
# 1. 加载伤寒论方剂数据集（简化示例）
dataset = [
    ['甘草', '桂枝', '麻黄', '杏仁'],
    ['大枣', '甘草', '桂枝', '生姜', '芍药'],
    ['甘草', '干姜', '附子'],
    ['桂枝', '芍药', '生姜', '大枣', '甘草', '饴糖'],
    ['大黄', '桂枝', '甘草', '桃仁', '芒硝']
]
# 2. 数据转换：将列表转换为 One-Hot 编码格式
te = TransactionEncoder()
te_ary = te.fit(dataset).transform(dataset)
df = pd.DataFrame(te_ary, columns=te.columns_)
# 3. 使用 Apriori 算法挖掘频繁项集 (最小支持度 0.6)
frequent_itemsets = apriori(df, min_support=0.6, use_colnames=True)
frequent_itemsets.sort_values(by='support', ascending=False).head(5)

# 4. 生成强关联规则 (最小置信度 0.8)
rules = association_rules(frequent_itemsets, metric="confidence", min_threshold=0.8)
